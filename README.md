### Wprowadzenie

Implementacja popularnej gry Tetris w języku Lua przy wykorzystaniu biblioteki LÖVE. Dźwięki do gry pochodzą ze strony https://soundbible.com/

### Wymagania

- Lua >= 5.4.4 (https://lua.org/)
- LÖVE >= 11.5 (https://love2d.org/)

### Uruchomienie

```sh
love src
```
